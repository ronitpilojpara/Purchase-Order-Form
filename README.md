# PURCHASE ORDER FORM
A simple and responsive Purchase Order Form built using React.js, HTML, CSS, and Bootstrap.
This project allows users to create, edit, and manage purchase orders in a clean UI.


# HOW TO CREATE AND RUN A REACT.JS APP
1. INSTALL PREREQUISITES
node -v
npm -v

2. CREATE A NEW REACT PROJECT
npx create-react-app my-app

3. GO INSIDE YOUR PROJECT FOLDER
cd my-app

4. START THE DEVELOPMENT SERVER
npm start


# PROJECT STRUCTURE

  App.css

# USAGE
1. Enter supplier and buyer details
2. Add items (description, quantity, price)
3. Total amount updates automatically
4. Submit or reset the form
5. Easily modify the form for your business logic


# FEATURES
1. User-friendly purchase order form
2. Add / Remove dynamic items
3. Auto-calculation of totals
4. Responsive design using Bootstrap
5. Clean component structure with React
6. Form validation
7. Easy to customize
